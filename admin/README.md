# Admin Management System

This project is an Admin Management System designed for handling menu items, processing orders, managing users, and providing an admin dashboard for easy navigation and management.

## Features

- **Menu Management**: Add, remove, and edit food items in the menu.
- **Order Processing**: Confirm orders and set statuses such as "in delivery" and "delivered".
- **User Management**: Manage user accounts and permissions.
- **Admin Dashboard**: A comprehensive dashboard displaying key statistics, recent orders, and top products.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd admin-management-system
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Start the development server:
   ```
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000`.
